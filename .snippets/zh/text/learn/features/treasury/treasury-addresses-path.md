## 财政部提案的路线图 {: #roadmap-of-a-treasury-proposal }

财政部支出请求的理想路径如下：

1. **提案提交** - 用户向 [Moonbeam 论坛](https://forum.moonbeam.network/c/governance/treasury-proposals/8){target=\_blank} 提交提案

2. **论坛讨论** - 社区在论坛上讨论该提案。最终的赞成/反对决定由财政委员会决定

3. **财政部批准和行动** - 如果财政委员会同意，它将授权财政支出并推进流程
