# 如何提出国库支出提案

## 简介 {: #introduction }

国库支出提案允许社区成员为有益于 Moonbeam 网络的项目申请资金，例如基础设施改进、资源、活动和生态系统工具。国库委员会审查提案并权衡在 Moonbeam 论坛收到的社区反馈。 但是，最终的赞成/否决决定权在于委员会。 有关国库委员会结构的更多信息，请参见[国库页面](/learn/features/treasury/){target=\_blank}。

在本指南中，您将学习如何准备和提交国库支出提案，并了解国库提案的完整生命周期。

## 定义 {: #definitions }

本指南的一些关键参数如下：

 - **国库地址** — 国库资金累积和支付的地址
 - **受益人** — 将收到国库提案资金的地址，例如 [Moonbeam Safe 多重签名](/tokens/manage/multisig-safe/){target=\_blank}
 - **价值** — 请求的金额，如果国库提案获得通过，将分配给受益人地址

## 向论坛提交您的想法 {: #submitting-your-idea-to-the-forum }

正如上述顺利流程中所述，国库提案的第一步是将提案提交到 [Moonbeam 论坛](https://forum.moonbeam.network/c/governance/treasury-proposals/8){target=\_blank}。您需要在 [Moonbeam 论坛](https://forum.moonbeam.network/){target=\_blank} 上拥有一个帐户并登录后才能提交国库支出提案。要提交国库支出提案，请按照以下步骤操作：

1. 在 [**治理** 部分](https://forum.moonbeam.network/c/governance/treasury-proposals/8){target=\_blank} 中，单击 **新主题**。通过在 **治理** 部分中启动主题，提案将预先填充一个模板，以确保您涵盖所有必要的要点
2. 提供提案的**标题**
3. 输入提案的内容，包括：**标题** 和 **提案状态**、简短的 **摘要**、**动机**、**项目概述** 和 **团队经验**、**理由**、任何 **关键术语**、**总体成本**、**国库资金的使用**、**技术规范** 和 **实施步骤**
4. 选择 **Moonbeam** 或 **Moonriver** 作为标签，以表明您的国库提案适用于哪个网络
5. 按 **创建主题**

## 后续步骤 {: #next-steps }

提出支出后，无需执行任何后续步骤。国库委员会的成员将完成所有链上操作。
如果获得国库委员会的批准，国库付款将自动交付给指定的受益人。有关国库委员会投票过程的更多信息，请参见[国库页面](/learn/features/treasury/#Treasury-council-voting-process){target=\_blank}。
