- **投票** — 一种代币持有者支持 (赞成)、反对 (反对) 或对提案保持中立 (弃权) 的机制。对于赞成和反对，投票权重由锁定的代币数量和锁定持续时间（信念）决定。弃权票不获得额外的权重。
    - **信念** — 代币持有者在投票时自愿锁定其代币的时间；锁定的时间越长，其投票的权重越大
    - **锁定余额** — 用户承诺用于投票的代币数量（注意，这与用户的总账户余额不同）
    Moonbeam 使用自愿锁定的概念，允许代币持有者通过将代币锁定更长时间来增加其投票权。不指定锁定周期意味着用户的投票价值为其锁定余额的 10%。指定更大的信念会增加投票权。对于信念（投票乘数）的每次增加，锁定周期都会翻倍
