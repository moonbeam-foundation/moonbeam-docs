- **主权账户** — 生态系统中每个链都有一个账户，一个用于中继链，另一个用于其他平行链。它通过将特定单词和和平行链 ID 连接起来的 `blake2` 哈希计算得出（中继链中的主权账户为 `blake2(para+ParachainID)`，其他平行链中的主权账户为 `blake2(sibl+ParachainID)`），并将哈希截断到正确的长度。该账户由 root 拥有，只能通过 SUDO（如果可用）或[治理（全民公投）](learn/features/governance/){target=_blank}使用。主权账户通常在生态系统中的其他链中签署 XCM 消息
 - **多重定位** — 一种指定相对于给定原点的整个中继链/平行链生态系统中的点的方式。例如，它可以用于指定特定的平行链、资产、账户，甚至平行链中的 pallet。一般来说，多重定位由 `parents` 和 `interior` 定义：
 
    - `parents` - 指的是从给定的原点进入父区块链需要采取多少个“跳”
    - `interior` - 指的是定义目标点需要多少个字段。
    
    例如，要从另一个平行链定位 ID 为 `1000` 的平行链，多重定位将是 `{ "parents": 1, "interior": { "X1": [{ "Parachain": 1000 }]}}`
