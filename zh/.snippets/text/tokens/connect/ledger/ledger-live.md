## 使用 Ledger Live 发送和接收 GLMR {: #use-ledger-live }

您还可以使用您的 Ledger 设备，从 Ledger Live 中安全地发送和接收 GLMR 代币。这使您无需将设备连接到 MetaMask 即可管理您的 GLMR 代币。

当您打开 Ledger Live 应用程序时，请确保您已安装最新的更新。如果有任何待安装的更新，屏幕顶部会出现一个横幅，提示您安装更新。

首先，您需要登录到您的 Ledger 设备以将其解锁。在 Ledger Live 中，点击 **My Ledger**（我的 Ledger）。在您的设备上，系统将提示您允许 Ledger 管理器；您可以点击设备上的两个按钮来允许它。

进入 Ledger 管理器后，您需要确保您的固件是最新的，并且如果 Moonbeam 和/或 Ethereum 应用程序需要更新，请继续安装最新版本。

接下来，您需要将帐户添加到您的 Ledger Live 应用程序。为此，您可以采取以下步骤：

1. 点击左侧菜单中的 **Accounts**（帐户）
2. 选择 **Add account**（添加帐户）
3. 将出现一个下拉菜单。搜索 GLMR，您应该会看到 **Moonbeam (GLMR)** 以供选择
4. 点击 **Continue**（继续）

![将帐户添加到 Ledger Live](/images/tokens/connect/ledger/ledger-live-1.webp)

接下来，您将能够输入帐户名称并点击 **Add account**（添加帐户）。如果您的帐户已成功添加，您可以点击 **Done**（完成），您的帐户将出现在您的帐户列表中。

### 接收代币

要将 GLMR 接收到您的 Ledger 设备，您可以从 Ledger Live 执行以下步骤：

1. 从左侧菜单中单击**接收**
2. 将出现一个弹出窗口。您可以从**记入账户**下拉列表中选择您想要接收代币的 Moonbeam 账户
3. 单击**继续**

![在 Ledger Live 中验证接收地址](/images/tokens/connect/ledger/ledger-live-2.webp)

接下来，您的地址应该会出现在 Ledger Live 上，并且系统会提示您在 Ledger 设备上验证您的地址。在您的设备上，您可以执行以下步骤：

1. 您应该在设备的屏幕上看到**验证地址**。单击右侧按钮以开始验证您的地址
2. 在下一个屏幕上，您应该会看到您的地址。将设备上的地址与 Ledger Live 上显示的地址进行比较，并验证它们是否匹配。此时，您需要从 Ledger Live 复制该地址，以便您可以向其发送交易。单击右侧按钮以继续
3. 现在，您应该会看到**批准**屏幕。如果地址匹配，您可以单击设备上的两个按钮以批准验证。否则，再次单击右侧按钮以进入**拒绝**屏幕，您可以在其中单击两个按钮以拒绝验证

![在 Ledger 设备上验证接收地址](/images/tokens/connect/ledger/ledger-live-3.webp)

在 Ledger Live 上，您会看到您的地址已安全共享，您可以单击**完成**。现在，您可以将一些 GLMR 发送到您的 Ledger 账户。

### 发送代币

要从您的 Ledger 设备发送 GLMR，您可以从 Ledger Live 执行以下步骤：

1. 点击左侧菜单中的**发送**
2. 将出现一个弹出窗口。从**扣款账户**下拉菜单中，您可以选择要从中发送代币的 Moonbeam 账户
3. 在**接收人地址**字段中输入地址
4. 点击**继续**

![在 Ledger Live 中发送交易](/images/tokens/connect/ledger/ledger-live-4.webp)

在下一个屏幕上，您可以输入要发送的 GLMR 数量，然后点击**继续**。

![在 Ledger Live 中输入要发送的数量](/images/tokens/connect/ledger/ledger-live-5.webp)

Ledger Live 上的最后一步是验证交易详细信息是否正确。如果一切正常，您可以点击**继续**。然后，系统会提示您在 Ledger 设备上确认交易：

1. 第一个屏幕将是**复核交易**屏幕。点击右侧按钮以前往下一个屏幕
2. 验证您要发送的 GLMR 数量，然后点击右侧按钮以继续
3. 验证您要将 GLMR 发送到的地址，然后点击右侧按钮以继续
4. **网络**屏幕应显示 **Moonbeam**。点击右侧按钮以继续
5. 查看**最高费用**，然后点击右侧按钮以继续
6. 如果一切正常，您可以点击两个按钮来**接受并发送**交易。否则，您可以点击右侧按钮进入**拒绝**屏幕，您可以在其中点击两个按钮来拒绝交易

![从 Ledger 设备发送交易](/images/tokens/connect/ledger/ledger-live-6.webp)

在 Ledger Live 上，您应该看到您的交易已发送，并且您可以查看交易的详细信息。交易确认后，您的 GLMR 余额将会更新。

您已成功使用 Moonbeam Ledger Live 集成，通过 Ledger 设备直接从 Ledger Live 接收和发送代币。
